Complex Function Plotter

Background

Most graphing calculators take a set of real numbers on the $x$-axis, run them through a function, and plot the result on the $y$-axis. The result we obtained is a two dimensional graph, as shown in the image below.

Complex numbers are an extension of the real numbers and are usually written in the form $x+y \,i,$ where $x$ and $y$ are real numbers and $i= \sqrt{-1}$ is the imaginary part. A complex function takes a complex number as an input and outputs a complex number. The input itself is two dimensional (real and imaginary) and so is the output, so four dimensions would be needed. Therefore a graphical depiction of complex functions such as in the real case is not feasible. To get around this, the graph here is shown in three dimensions and color is used to encode the fourth dimension needed.

How to use

Start by typing an equation in the textbox in the upper-left. It should be in the form f(z)=..., where the function takes a complex number and does something to it. If the function is valid, the 3D graph should update.

The height of the graph defaults to the absolute value of the output of the current function at every input. By default, the hue of a point equals its angle (in polar coordinates).

To rotate the graph, click and drag or touch and drag with one finger (on a touch screen). Use the mouse wheel or two fingers to zoom in or out. Right click and drag or drag with two fingers to pan the view.

Note: This program works best in FireFox and Chrome. It may work in Edge, but does not usually work in Internet Explorer.

Some things to try

The function $f(z)=z$ is good to start with. Polynomials are always neat. For example $$ f(z)=z^2+1,\quad f(z) = 2z^3+3z^2+z-1, \quad f(z) = z^5-1. $$ You can clearly see that a polynomial always has as many zeros as the exponent on its highest term (some may be in the same position).

The function $f(z)=z^c$ is also cool, with the slider enabled. The functions $$f(z)=(((z^2+z)^2+z)^2+z)^2 \quad \text{and}\quad f(z)=z{^{z^{z^{z^z}}}}$$ both have much fine detail, so are best viewed with height as flat. The functions $\sin(z),$ $\cos(z),$ $\tan(z),$ $\text{gamma}(z),$ and $\text{zeta}(z)$ are also interesting to explore in 3D. For example type f(z)=gamma(z).

About

This program was originally created by David Brock. The source code is available in replit.

This version uses the color scheme known as enhanced phase protrait to include level curves of the absolute value of the output and the angle (modulus and phase/argument). I also added more options for the 3D plots to graph the real and imaginary components.

How this works

The real power of this program is that it can turn any equation into OpenGL code that can be run on a GPU and plotted on the screen. The programm uses Three.js for the 3D graphics. It also uses Math.js to compile the equation into an expression tree and then replace each node with the correct OpenGL function, using 2D vectors for all complex numbers. Almost every function (except addition) had to be written to accept vectors as complex numbers.

If you see "grass/noise" extending off the screen, this usually means the value of the function is extremely large or chaotic. Also, since the plot is generated on the GPU, it is calculated in single precision (about 8 digits) while the plot cursor is calculated in double precision (about 16 digits), so this can lead to slight differences.

Available Variables and Functions

z This is the input to the function being plotted.
x, y These are the real and imaginary parts of z.
i The imaginary unit. i^2=-1
pi 3.14159265...
e 2.718281828...
c This is a constant. It can be adjusted with a slider, if enabled in options.

re The real part of a number
im The imaginary part of a number
conj The conjugate
abs The absolute value, or magnitude (abs(z)=sqrt(re(z)^2+im(z)^2))
polar The number in polar coordinates. The real part is the magnitude and the imaginary part is the angle (in radians).
ln Natural log, or loge. For complex numbers the result is similar to polar coordinates.
exp Raise e to a power (exp(z)=e^z)
sqrt Square root (sqrt(z)=z^0.5)
sin, cos, tan, csc, sec, cot Trigonometric functions
asin, acos, atan Inverse trigonometric functions
sinh, cosh, tanh Hyperbolic trigonometric functions
asinh, acosh, atanh Inverse hyperbolic trigonometric functions
gamma The gamma function. It can be thought of an an extension of factorials.
zeta The Riemann zeta function. With a low number of iterations a line is visible at re(z)=0.5 due to the way it is calculated.

Constant c

This is a number you can use in equations (like pi, e, etc.). You can change it by clicking the line "c = ...", or using sliders if enabled with the checkbox in Options. The advantage of using c instead of defining a constant in the equation box is that it can be changed without the equation needung to be recompiled.

×

3D Plot

This is the actual plot of your function. You can change what the Height and Color mean.

Height options:

  • abs(z): Modulus of $f(z)$.
  • re(z): Real component of $f(z)$.
  • im(z): Imaginary component of $f(z)$.
  • Flat: 2D plot of $f(z)$.

Color options:

  • Simple arg(z): Angle of $f(z)$.
  • Enhanced arg(z): Level curves of the angle of $f(z)$.
  • Enhanced abs(z): Level curves of the modulus of $f(z)$.
  • Enhanced abs(z) & arg(z): Both level curves of the angle and modulus of $f(z)$.
  • Enhanced re(z):Level curves of the real components of $f(z)$.
  • Enhanced im(z): Level curves of the imaginary components of $f(z)$.
  • Enhanced re(z) & im(z): Both level curves of the real and imaginary components of $f(z)$.
×

Input Plane

This is the set of inputs to your function. It is analogous to the X axid on a traditional graphing calculator. It is a portion of the complex plane with the real and imaginary axes numbered.

×

Plot Cursor

This allows you to inspect the plot more closely. If you click (or mouse over) a point on the input plane, you can see the exact input and output of the function at that point. The "Continuous" option toggles whether the cursor only moves when you click. Un-checking this is useful if an equation takes too long to recalculate or you want to watch a specific point.
