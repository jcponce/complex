Mandelbrot and Julia Sets

Welcome! With this application you can explore the relationship between the Mandelbrot and Julia Sets. Move the mouse around to see different Julia sets.

Open the controls (upper right corner) to change the number of iterations or to set the value of c.

Control keys

: Up

: Down

: Right

: Left

: Zoom In

: Zoom Out

R : Reset

I : Info - Center coordinates
