On the derivative of the complex power function

The generalized complex power function is defined as: $$w = z^c = \exp(c\log(z)), \quad \text{with}\quad z \quad \text{not equal to 0}.$$

Due to the multi-valued nature of $\log(z)$, it follows that $w=z^c$ es also multi-valued for any non-integer value of $c$, with a branch point at $z=0$: $$w=z^c=\exp(c \log(z))=\exp(c \,\text{Log}(z))\exp(2 \pi n c i),$$ with $n=\ldots-2,-1,0,1,2,\ldots$

On the other hand, we have that the generalized exponential function, for c not equal to 0, is defined as: $$w=c^z=\exp(z\log(c))=\exp(z (\text{Log}(c)+2 \pi n i)),$$ with $n=\ldots-2,-1,0,1,2,\ldots$

However, the multi-valued nature of this function differs somewhat from the multi-valued generalized complex power function. The generalized exponential function possesses no branch point (or any other type of singularity) in the infinite complex z-plane. Thus, one can regard the equation: $$w=c^z=\exp(z\log(c))=\exp(z(\text{Log}(c)+2\pi n i)),$$ with $n=\ldots-2,-1,0,1,2,\ldots$

As defining a set of independent single-value functions for each value of $n$. Typically, the $n=0$ case is the most useful, in which case, we would simply define: $$w=c^z=\exp(z\log(c))=\exp(z(\text{Log}(c)),$$ with $c$ not equal to zero.

This conforms with the definition of exponential function $$e^z=e^x(\cos(y)+i\sin(y))$$ where $c = e$ (the Euler constant).

As a final remark: In practice, many textbooks treat the generalized exponential function as a single-valued function, $c^z=\exp(z\,\text{Log}(c))$, only when $c$ is a positive real number. For any other value of $c$, the multi-valued function $c^z=\exp(z \log(c))$ is preferred.
